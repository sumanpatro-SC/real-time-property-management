# Real-Time Property Management Platform

## 🚀 Overview
A full-stack web application to manage rental properties, maintenance requests, and amenity bookings in real-time.

## ✨ Features
- Real-time maintenance tracking
- Amenity booking system with conflict prevention
- Role-based dashboards (Tenant / Owner / Admin)
- Live status updates using Socket.IO

## 🛠 Tech Stack
- Frontend: React / Next.js
- Backend: Node.js + Express
- Database: MongoDB
- Realtime: Socket.IO
